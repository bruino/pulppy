Pulppy

Linear Programming software for optimizing various practical problems of Operations Research.

  • Download ZIP
  • Download TAR
  • View On GitHub

This project is maintained by bruino

Features

It aims to resolve linear models with a intuitive, simple and multi platform graphical interface that deals with practical problems of Operative Research of "Integer", "Binary", "Continuous" and "Mixed" types. The project has the following dependencies: PuLP, PyQt5, Clp, Matplotlib and Numpy.
